## Telegram [youtube-dl] bot
This is a simple bot that download youtube videos if you give it a url
To Use it you need to:
1. Install [youtube-dl], install instructions can be find [here](https://github.com/ytdl-org/youtube-dl/blob/master/README.md#installation)
2. Install ffmpeg if you want merge audio and video dwonloaded by [youtube-dl]
3. Create config.py and put your bot token in it, if you don't have a token, contact [bot-father](https://t.me/botfather)
